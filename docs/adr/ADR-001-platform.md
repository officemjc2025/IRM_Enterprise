# ADR-001 : IRM Platform Architecture

Status: Accepted

Date: 2026-06-26

---

## Decision

IRM จะถูกพัฒนาเป็น Multi-Project Platform

ไม่ใช่ระบบเฉพาะ Metro Jomtien

Metro Jomtien เป็นเพียง Reference Project แรก

---

## Reason

- รองรับหลายโครงการ
- Branding แยกตามโครงการ
- Deploy ได้หลาย Site
- ใช้ Codebase เดียว
